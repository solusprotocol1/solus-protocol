Security architecture

Defense-grade security design. Zero sensitive data on-chain. Classified-network ready.

Zero Data On-Chain

Only SHA-256 hashes are stored on the XRP Ledger. No CUI, no ITAR, no PII. The hash is irreversible — original data cannot be recovered.

NIST SP 800-171

Architecture aligned with all 14 control families. Access control, audit accountability, incident response, and system integrity built in.

CMMC Level 2

Designed to meet CMMC Level 2 requirements for handling CUI. Pursuing formal assessment on the FedRAMP authorization path.
